About Extension Dev

Extension Dev is a platform for developers to create and share extensions for Replit. With our simple API and easy-to-use interface, anyone can create an extension and share it with the world.

How to use Extensions

To use an extension, simply navigate to the Extensions tab and click the "Add Extension" button. From there, you can browse and install any available extensions.
